# Real-Time Chat Application

A real-time chat application built using Node.js, Express, Socket.IO, and SQLite.

## Features

* Public Chat
* Private Messaging
* Online Users List
* Typing Indicators
* Unread Message Badges
* Public Chat Unread Count
* Message Timestamps
* SQLite Message Persistence
* Persistent User Identity
* Private Chat History Reload

## Tech Stack

* Node.js
* Express.js
* Socket.IO
* SQLite
* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
node app.js
```

4. Open browser

```text
http://localhost:3000
```

## Project Structure

* app.js - Server-side application
* public/ - Frontend files
* chat.db - SQLite database
* package.json - Project dependencies
